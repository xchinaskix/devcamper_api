# devcamper api

> backend api for api devcamper application which is a bootcamp directory websitte

## usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## install dependencies
```
npm install
```

## run app
```
# run in dev mode
npm run dev

# run in prod mode
npm start
```
